### gulp.js
Streaming build system.

``npm install --save-dev [pkg]``

where ``[pkg]`` indicates a package that will be used in ``gulpfile.js``.

Uses the following packages:

* ``gulp``
* ``gulp-jshint``
* ``gulp-concat``
* ``gulp-uglify``
* ``gulp-rename``
* ``gulp-connect``
* ``gulp-open``
